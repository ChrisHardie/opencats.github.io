Documentation

The following resources are available online for users and developers of opencats:

Users

Installation Instructions

opencats is currently under development, and no releases have been published yet. To install a pre-alpha, experimental version of opencats, please refer to the installation instructions in the opencats Wiki.

User Manual

opencats's manual is not yet available online. In the meantime, you can use the Vim documentation instead. If you have installed opencats, you can also execute the :help command to access the manual.

Quick Reference

opencats (and Vim) ships with a quick reference guide: :help quickref

Vim FAQ

Developers & Contributors

opencats Wiki

opencats's wiki contains a number of links, hints and guides for opencats users and developers.

Developer Documentation

Visit the developer documentation page for an up-to-date Doxygen documentation of opencats's code.

Clang Report

Clang's static analyzer is executed periodically on the opencats codebase. A list of current issues is published in the static analysis report.

Coverity Report

A list of issues found by Coverity is available at the opencats Coverity project. To get access, have a look at the hints on the opencats wiki.

Translation Report

An overview of the current status of translations in opencats is published on the translation report page.
